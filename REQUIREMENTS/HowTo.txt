# How to install the requirements?

Step 1:
```
Download the requirements.txt file
```

Step 2:
```
Open the terminal , change directory to the folder where requirements.txt is , then run:

pip install -r requirements.txt
  
```

Voilla!!! You are good to go!

If you do not have python installed , then:

For Ubuntu users : https://www.geeksforgeeks.org/how-to-install-python-in-ubuntu/

For Windows users : https://www.digitalocean.com/community/tutorials/install-python-windows-10
